# Mindset

- Prioritize maintainable, extendable, modular design so swapping third-party
  services, deploy patterns, or UIs (mobile/web/Telegram/etc.) is easy.
- Keep code as simple as possible.
- Write tests for every module and struct to ensure other parts can use them
  correctly.
